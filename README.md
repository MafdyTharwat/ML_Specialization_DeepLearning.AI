# Machine Learning Specialization — DeepLearning.AI & Stanford University

Welcome to my repository containing code implementations, concise notes, and practical labs completed as part of the **Machine Learning Specialization**, taught by **Andrew Ng**.

This 3-course series provides a rigorous foundational breakdown of modern Machine Learning techniques, ranging from fundamental statistical models to Deep Learning and Reinforcement Learning.

---

## Specialization Overview

### Course 1: Supervised Machine Learning (Regression and Classification)
* **Linear Regression:** Cost function, Gradient Descent algorithm, Vectorization with NumPy, Feature Scaling.
* **Logistic Regression:** Classification decision boundaries, Sigmoid function, Cost function analysis.
* **Overfitting & Regularization:** L1 and L2 regularization techniques to prevent variance issues and improve model generalization.

### Course 2: Advanced Learning Algorithms
* **Neural Networks:** Multi-layer Perceptrons (MLPs), Forward Propagation, Backpropagation, Activation Functions (ReLU, Sigmoid, Softmax).
* **Model Evaluation & Diagnostics:** Train/Validation/Test split, Bias vs. Variance diagnosis, Error Analysis, Learning Curves.
* **Decision Trees & Ensembles:** Decision Trees, Random Forests, XGBoost, and feature importance.

### Course 3: Unsupervised Learning, Recommenders, & Reinforcement Learning
* **Unsupervised Learning:** K-Means Clustering, Anomaly Detection (Gaussian Distribution).
* **Recommender Systems:** Collaborative Filtering, Content-Based Filtering, Principal Component Analysis (PCA).
* **Reinforcement Learning:** Deep Q-Networks (DQN), Markov Decision Processes (MDP), State-Action-Reward cycles.
